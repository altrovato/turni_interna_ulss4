# Turni Reparto

App gratuita per i turni del reparto: PWA, cloud condiviso, login medici.

**Guida completa (semplice, per tutti):** [`GUIDA.md`](GUIDA.md)

---

## Per i medici

Apri il link del reparto → accedi con email e password.  
Tutto spiegato passo passo in **GUIDA.md** (entrata, ferie, consultazione turni).

## Per il primario / turnisti

Stessa guida: Parte 3 e 4 spiegano compilazione turni, weekend, bozza, Excel.

## Per chi installa online (tecnico)

Appendice in **GUIDA.md** + file `supabase/schema.sql` e `supabase/schema-v2-auth.sql`.

Servizi gratuiti: [Supabase](https://supabase.com) + [GitHub Pages](https://pages.github.com). Nessun dominio da comprare.
